# Interactive Dashboard Development

## Objective
Develop an interactive dashboard to visualize and analyze business data effectively.

## Dataset

| Month | Sales | Profit | Customers |
|--------|--------|--------|----------|
| Jan | 50000 | 10000 | 200 |
| Feb | 55000 | 12000 | 220 |
| Mar | 60000 | 15000 | 250 |
| Apr | 65000 | 17000 | 280 |
| May | 70000 | 20000 | 300 |

## Dashboard Features
- Sales Analysis
- Profit Analysis
- Customer Growth Tracking
- Interactive Charts and Graphs
- Data Filtering

## Analysis
- Sales show steady growth over time.
- Profit increases with sales growth.
- Customer count rises consistently.
- Dashboard helps identify trends quickly.

## Conclusion
Interactive dashboards provide real-time insights and improve decision-making through visual representation of data.
